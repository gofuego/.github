# Fuego

A **meta-engine for static site generation** in Go — you define the content
format, Fuego handles discovery, parsing, routing, taxonomies, collections,
rendering, and serving. No format is privileged; Markdown is just one opt-in parser.

## Projects

- **[fuego](https://github.com/gofuego/fuego)** — the engine.
- **[fuego-studio](https://github.com/gofuego/fuego-studio)** — authenticating host + in-place editor for Fuego sites.
- **[fuego-doctheme](https://github.com/gofuego/fuego-doctheme)** — shared documentation theme packs.
- **[fuego-docs](https://gofuego.github.io/fuego-docs/)** · **[fuego-studio-docs](https://gofuego.github.io/fuego-studio-docs/)** — the docs.
